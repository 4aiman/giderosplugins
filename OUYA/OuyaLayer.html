Implementing OUYALayer.lua

OUYALayer allows you to make any your click/touch scene objects to DPAD selectable Ouya compatible objects just use it as scene base class or layer on the scene

All you need to do is additionally to adding selectable objects as child or any level subchild of the OUYALayer, you additionally need to call OUYALayer:addObject(sprite) on every selectable object

Also as ouya object is global, all added events are kept with it, but adding ouya events to OUYALayer, it will automatically manage them for specific scenes/layers

Implement :back() method to ouya layer to be automatically call it uppon OUYA_A button press

Here is an example of OUYALayer in action: http://www.youtube.com/watch?v=q8uR9sAQg0g

API Reference

Methods

OUYALayer:addObject(sprite, id) - adds selectable objects (basically you should add any object that can be clicked), providing order id (first object is automatically selected)

OUYALayer:setFillStyle() - setting the fill style of the hover

OUYALayer:setLineStyle() - setting the line style of the hover

OUYALayer:setHoverActive(bool) - set true to enable default hover, false to disable it

OUYALayer:disable() - disable all the ouya specific functionality of the layer (when working with sublayers as using OUYALayer for popups)

OUYALayer:enable() - enable all the ouya specific functionality of the layer (enabled by default)

Events

"selected" - dispatched when any new object is selected and needs to be hovered

  • e.object - contains the reference to selected object

And all the other events of ouya object
